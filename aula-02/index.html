Sobre o curso

Nosso curso online de HTML é gratuito e foi projetado para ensinar aos iniciantes os conceitos básicos e avançados da linguagem de marcação. Durante o curso, você aprenderá a criar páginas da web usando HTML puro, bem como as melhores práticas para criar páginas da web acessíveis e responsivas.

O que você vai aprender

  • Os fundamentos do HTML, incluindo a sintaxe básica e as tags mais comuns.
  • Como criar formulários, tabelas e listas em HTML.
  • Como usar CSS para estilizar suas páginas da web.
  • Como criar páginas da web responsivas que funcionam em dispositivos móveis.
  • Como tornar suas páginas da web acessíveis para usuários com deficiência.

Benefícios do curso

  1. Acesso gratuito ao nosso curso online de HTML.
  2. Aprenda com instrutores experientes e altamente qualificados.
  3. Aprenda em seu próprio ritmo, no seu próprio horário.
  4. Certificado de conclusão após a conclusão do curso.
